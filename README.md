# ScriptCat 脚本仓库

这是一个存放 ScriptCat 用户脚本的仓库，收集了作者自己编写或搜寻而来的实用脚本。

## 什么是 ScriptCat？

ScriptCat 是一款功能强大的浏览器用户脚本管理器，支持 GreaseMonkey、TamperMonkey 等用户脚本标准。

官方网站：https://scriptcat.org/

## 脚本列表

## 安装方法

1. 安装 [ScriptCat 浏览器扩展](https://scriptcat.org/)
2. 克隆本仓库或下载所需脚本
3. 在 ScriptCat 中添加脚本即可使用
